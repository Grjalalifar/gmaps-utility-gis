Simple Drag Box Zoom

This lib provides a very simple drag zoom. Holding shift key while draw a box will zoom to the desired area. Only one line of code GMap2.enableDragZoom() is needed.

For a description and examples of how to use this library, check out the how-to.

class DragBoxZoomOptions

This class represents optional parameter passed into GMap2.enableDragBoxZoom(). There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
key String the combo key to use while draging the box, shift | alt | ctrl . Default is shift.
style Object javascript object used for zoom box css style. e.g. {backgroundColor:'blue',opacity:0.6, filter:'alpha(opacity = 50)'}

class GMap2

This is new methods added to Google Maps API's GMap2 class.

Methods

Methods Return Value Description
disableDragBoxZoom() None Disable drag zoom
dragBoxZoomEnabled() Boolean Returns true if the drag zoom is enabled.
enableDragBoxZoom(opt_dragZoomOpts:DragBoxZoomOptions) None Enable drag zoom. User can zoom to a point of interest by holding a special key (shift | ctrl | alt ) while drag a box.
